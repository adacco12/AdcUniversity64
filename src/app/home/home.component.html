This is an example of how a web application can solve the computerization of a simple university

The tools I used to make this project are: Angular 7, HTML, CSS, Bootstrap, Javascript, jQuery, Node js, jQWidgets, ngx-datatable, MongoDB, VB.net, C#, Ajax, Crystal Reports


Emphasis has been placed on different ways and methods of displaying, entering, modifying and analyzing data from databases.


The techniques used in this project can be used to solve any requirement regarding the computerization of any kind of records.


Prints on screen and paper are solved with the world's most powerful reporting tool Crystal Reports



If you want to see my simple university solution as a desktop application, you can download it from here: AdcDownload.exe
Both apps work with the same database.


I hope you like the way I make websites and desktop applications..
